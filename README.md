# BarberPoint API

API for the operation of the App and the WebSite that has the following resources:

  - DotNet Core 3.1
  - MediatR
  - Dapper
  - Identity Core
  - Dependency Injection
  - Authentication JWT
  - MySql
  - FluentMap
  - FluentValidation
